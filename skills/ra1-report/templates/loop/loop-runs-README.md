# Loop Runs

This directory records agent loop attempts. A run log documents evidence for maintainers; it is not RA1 loop-autonomy clearance.

## Convention

Create one directory per run: `loop-runs/YYYYMMDD-HHMMSS-slug/`.

Each run directory should contain:
- `README.md` with goal, operator, commands run, result, artifact paths, and follow-ups.
- `artifacts/` for screenshots, videos, logs, or generated reports when they are useful evidence.

RA1 checks that this convention exists. The loop runner is responsible for enforcing behavior.
