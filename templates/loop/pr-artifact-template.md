# PR Artifact Evidence Template

Use this checklist when preparing a PR that was produced or verified by a loop run.

## Evidence to cite

- CI run URL or local command transcript.
- Loop run log under `loop-runs/YYYYMMDD-HHMMSS-slug/README.md` when a loop runner was used.
- Screenshot or video for UI/mobile changes when behavior is visible.
- Logs, SARIF, coverage, or test output under `.agents/artifacts/` or `loop-runs/YYYYMMDD-HHMMSS-slug/artifacts/` when relevant.
- Any skipped verification with the reason and owner for follow-up.

RA1 checks that this evidence contract exists. It does not assert the artifacts prove behavior.
